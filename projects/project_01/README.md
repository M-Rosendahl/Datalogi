# Project 1 - Texas Department of Insurance (TDI) : All Data

The Texas Department of Insurance (TDI) handles complaints against people and organizations licensed by TDI, such as companies, agents, and adjusters.

This dataset includes a row for each person and organization named in a complaint. This means some complaint numbers are listed multiple times.
